# TETRIS
-----
This project is made using the pygame module in Python. 
-----
## Features of this game 
- Multi-Dimensional List Processing
- Game Loops
- Basic Pygame Graphics
- Scoring & Saving High Scores
- Increasing Game Speed and Difficulty
- End Screen
- Start Menu
